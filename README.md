# OSMChina-Localization
OpenStreetMap 简体中文本地化 索引

----------

本项目用于记载各类OSM社区内的可翻译内容，以及其可见的翻译平台位于何处。

以OSMChina名义提交了本地化内容的，我们将进行特殊注明。

## 基础内容

* OpenStreetMap 官网
* OpenStreetMap Wiki

## 编辑器类

* iDeditor/Rapid
* JOSM

## 工具类网站

* welcome.osm.be
* HOT
* Taginfo

## 衍生网站

* OpenRailwayMap

## 基础设施

* Unicode-cldr
